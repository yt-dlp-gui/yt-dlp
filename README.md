<center>
  <h1><b>GUI for yt-dlp</b></h1>
  
</center>

This project is a graphical user interface (GUI) developed entirely in PowerShell. It was created for strictly educational purposes to demonstrate the capabilities of Windows script automation and advanced interaction with command-line tools (CLI).

<center>
  <h2>📌 Project Objectives</h1>
  
</center>

This repository serves as a case study for:

PowerShell Automation: Using WinForms/WPF within .ps1 scripts.

Process Management: Interoperability between PowerShell and external binaries such as yt-dlp and ffmpeg.

Metadata Research: Technical analysis of multimedia data flows for development environments.
